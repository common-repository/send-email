Send email allows you to send email from your WordPress dashboard conveniently. 

== Description ==

Send email directly from your WordPress dashboard. No need to visit your mailbox now to create an email. Send your business or personal emails directly from WordPress admin email address and easily. 


== Installation ==

* Upload the Send email plugin to the `/wp-content/plugins/` directory
* Activate the Send email plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= How do I use Send Email plugin? =
* The link to access the tool 'Send email' is in the "Tools" menu.

= Haven't received email? =
* Email may be in spam folder. 
* If your Host and WordPress email functionality is working, it works for sure. 

= How long it take email to go? =
* Email goes immediately within few seconds.
